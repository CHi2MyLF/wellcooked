# WellCooked
//暂时想不到更好的名字了

---
## 产品背景

冰箱里一到过年（实际上平日也是）塞满一堆菜，吃腻了或是不会做。
于是Trae控制大脑，AI代替思考

---

## 核心功能

- 🎙️ **语音输入食材** — 说出你手边的食材，免得手上沾水，不想碰屏幕
- 🍽️ **AI生成菜谱** — 以你输入的食材为主角，生成可执行的菜谱
- 📖 **菜谱记录** — 保存生成过的菜谱，随时回看（只保留了十条）
- ✅ **标记菜谱** — 灵感来源于某瓣（special thanks）
